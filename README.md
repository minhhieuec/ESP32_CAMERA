# ESP32_CAMERA
## SET PATH
### IDF_PATH
1. edit file <b>idf_cmd_init.bat</b> (comment line 9 to REM set IDF_PATH=%CD%, or set IDF_PATH=...).
2. open Edit Environment Variables and add <b>IDF_PATH</b>.
